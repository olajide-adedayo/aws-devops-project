# AWS DevOps Project - EC2 & Nginx Deployment

## Project Overview
This project demonstrates how to deploy a web server on AWS using EC2 and Nginx.

## Steps Completed
- Launched AWS EC2 instance (Ubuntu 24.04 LTS)
- Configured Security Group (SSH & HTTP access)
- Installed and started Nginx web server
- Verified deployment using public IP

## Technologies Used
- AWS EC2
- Ubuntu Linux
- Nginx
- GitHub
